# メドローア — 極大消滅呪文シミュレーター

ハンドジェスチャーでメドローアを撃てるWebアプリ。
Webカメラの映像に手検出（MediaPipe Hand Landmarker）を掛け、Canvas 2Dでエフェクトを合成しています。サーバーサイド処理はなく、映像は端末外に送信されません。

## 遊び方

1. **右手を開く** → 🔥 メラゾーマをチャージ
2. **左手を開く** → ❄️ マヒャドをチャージ
3. **両手を合わせる** → ⚡ 融合（ゲージが溜まるまでキープ）
4. **両手をグーにして片手を引く** → 🏹 光の弓矢が形成され、引くほど引き絞りゲージが上昇
5. **引いた手をパーに開く** → **メドローア発射！！**
   - 矢は「引き手→弓手」の延長方向へ飛びます（斜め撃ちも可能）
   - 引き絞りが3割未満だと発射されず、融合状態に戻ります

## 隠しコマンド 🤫

チャージのとき、両手とも**パーではなく「指2本（人差し指＋中指）」**で構えると、
魔法力を調整してメドローアに似せた**ベギラマ**になります（原作のシグマ戦のフェイク再現）。

- エフェクトがフラットな黄色一色のシンプルな見た目に変わります（チャージ中の呪文名表示も消えます）
- 発射時のセリフ：**「横っ面を…ひっぱたくんだよおっ!!!」**
- さらに、この直後に本物のメドローアを構えて**弓を引いた瞬間**に **「化かし合いは…俺の勝ちだッ!!」**、射出時はルビなしの **「メドローア！！」** のみ表示されます（原作の決着シーン再現）
- 発射までの流れ（合わせる→グーで引く→パーで発射）は本物と同じです

## 起動方法

```sh
cd このフォルダ
python3 -m http.server 8765
# → http://localhost:8765 をブラウザで開く
```

Chromeなら `index.html` をダブルクリック（file://）でも動作します。
手検出モデルのダウンロードに初回のみインターネット接続が必要です。

## 公開用ファイル

公開URL: https://stella-create.co.jp/tools/medoroa/

- `index.html` — アプリ本体（OGP・ファビコンのメタタグ設定済み）

## 技術構成

- アプリ本体は単一HTMLファイル（`index.html`）
- 手検出: [MediaPipe Tasks Vision — HandLandmarker](https://developers.google.com/mediapipe/solutions/vision/hand_landmarker)（CDN・GPU委譲、失敗時はCPUにフォールバック）
- エフェクト: Canvas 2D（加算合成パーティクル、ビーム、画面シェイク）
- 効果音: Web Audio API による完全合成（外部音源ファイル不要）
- 手の左右判定はミラー表示上の位置ベース（画面右側の手＝炎）なので、検出モデルの左右ラベルに依存せず頑健
